## 修复 Button 组件 ref 转发警告
- **问题**: `DialogPrimitive.Close` 用 `render={<Button />}` 时向 Button 传 ref，但 Button 是普通函数组件不支持 ref 转发，触发 React 警告
- **修复**: `frontend/src/components/ui/button.tsx` 中用 `React.forwardRef` 包装 Button，将 ref 转发给底层 `ButtonPrimitive`
- **验证**: tsc --noEmit 通过，无类型错误

### 部署
需要重建 Docker 镜像：`docker compose build backend && docker compose up -d backend`

## GitHub Actions CI/CD 配置

- 创建 `.github/workflows/deploy.yml`，push 到 main 后自动 SSH 到服务器执行 `git pull + docker compose build + up -d`
- 需要用户在 GitHub 仓库配置 3 个 Secrets：`SERVER_HOST`、`SERVER_USER`、`SERVER_SSH_KEY`

## GitHub Actions 部署排查
- push 成功但 Actions 报 exit code 1，根因：deploy.yml 写的是 `git pull origin main`，实际分支是 `master`，已修复
- Secrets 配置方式从 SSH key 改为 password，因为用户用的是密码登录

## 新增 5 个基础工具 (2026-07-11)
- **get_news**: 微博热搜榜单，API: api.vvhan.com（免费）
- **lookup_ip**: IP 归属地 + 运营商查询，API: ip-api.com（免费）
